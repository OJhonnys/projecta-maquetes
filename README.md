Projecta Maquetes – Website Redesign
Sobre o projeto

Este projeto consiste no redesign do website da empresa Projecta Maquetes, com o objetivo de modernizar a interface, melhorar a experiência do usuário e facilitar o contato com clientes interessados nos serviços da empresa.

O novo site apresenta um design mais limpo, navegação simples e integração direta com ferramentas de comunicação e análise de dados.

Tecnologias utilizadas

HTML5

CSS3

JavaScript

Funcionalidades

Redesign completo do website institucional

Interface moderna e responsiva

Integração com API do WhatsApp para contato direto com a empresa

Implementação do Google Analytics para monitoramento de acessos e comportamento dos usuários

Estrutura otimizada para melhor navegação e experiência do usuário

Integrações
WhatsApp API

O site possui um botão de contato que redireciona o usuário diretamente para uma conversa no WhatsApp da empresa, facilitando o atendimento e o envio rápido de solicitações de orçamento.

Google Analytics

Foi implementado o Google Analytics no código do site para coletar dados sobre:

número de visitantes

páginas mais acessadas

comportamento dos usuários

origem do tráfego

Essas informações ajudam na tomada de decisões estratégicas e melhorias contínuas do website.

Objetivo do projeto

O objetivo principal é fortalecer a presença digital da empresa, tornando o site um canal eficiente de apresentação dos serviços e geração de novos contatos comerciais.
