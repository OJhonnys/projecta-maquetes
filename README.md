# Projecta Maquetes – Website Redesign

Redesign of the official website for Projecta Maquetes, focusing on improving visual identity, user experience, and the company's digital presence.

The project is developed using React, JavaScript, and CSS, creating a modern and responsive interface. The website also integrates communication and analytics tools to improve interaction with potential clients and monitor website performance.

Features

Modern and responsive design

Component-based architecture using React

Styling implemented directly with CSS

WhatsApp API integration for direct contact with clients

Google Analytics integration for traffic monitoring and user behavior analysis

Technologies

React

JavaScript

CSS

WhatsApp API

Google Analytics
